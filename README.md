## How to Run

### 1. Clone Repository

```bash
git clone https://github.com/your-username/project-name.git
cd project-name
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Set Up Database

* Install PostgreSQL
* Create a database
* Update connection details in the Python script

### 4. Run Python Scripts

```bash
python data_loading.py
python eda_analysis.py
python data_cleaning.py
```

### 5. Execute SQL Queries

* Open PostgreSQL client (pgAdmin / psql)
* Run queries from `/sql` folder

### 6. Open Power BI Dashboard

* Open `.pbix` file in Power BI Desktop
* Refresh data connection

### 7. View Report & Presentation

* Report available in `/reports`
* PPT created using Gamma in `/presentation`
